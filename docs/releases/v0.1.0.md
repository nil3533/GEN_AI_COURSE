# v0.1.0 - Spring AI Foundations, Streaming, and Tool Calling

This release packages the first four completed modules of Spring AI GenAI Journey.

## Included Modules

| Module | Topic | Mini-project |
|---|---|---|
| 01 | Foundations of GenAI | Raw LLM HTTP API with Spring `RestClient` |
| 02 | Spring AI Core | Multi-provider ChatClient service |
| 03 | ChatClient Deep Dive | SSE streaming chat API |
| 04 | Structured Output and Tools | Order assistant with backend tools |

## Highlights

- Java 21 and Spring Boot 3.5.x.
- Spring AI provider switching with Groq and Ollama.
- Streaming and non-streaming chat endpoints.
- Structured output with typed Java records.
- `@Tool` backend method exposure.
- Tool traces and cancellation safety checks.
- Deterministic tests that avoid live LLM calls by default.
