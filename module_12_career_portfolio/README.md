## Mini-project

**Portfolio polish pass.**

1. Rewrite resume using file 02's patterns. Have one trusted person review.
2. Polish every project README in `sani-genai-journey` — problem statement, architecture diagram, run instructions, screenshots, "what I learned."
3. Deploy at least 2 projects to free hosting (Cloudflare Pages for React frontends; a free tier for Spring Boot apps — Koyeb, Fly.io, Railway).
4. Write 2 technical blog posts on Medium/Dev.to. Suggested topics: "Building production RAG in Spring Boot with pgvector," "Spring AI Advisors are AOP for LLM calls."
5. Record a 2-minute Loom walking through your capstone (M13) — link from your resume and LinkedIn.
6. Update LinkedIn with the new positioning. Connect with 30 Indian Spring AI / AI Engineer practitioners.
